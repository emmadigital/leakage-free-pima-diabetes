# Leakage-Free Diabetes Prediction

This repository contains MATLAB code used for the study:

Leakage-Free and Reproducible Evaluation of Neural Networks for Diabetes Prediction.

## Models Evaluated

Baseline Models
- Logistic Regression
- Random Forest

Neural Network Models
- Feedforward Neural Network (FFNN)
- Backpropagation Neural Network (BPNN)
- Generalized Regression Neural Network (GRNN)
- Artificial Bee Colony optimized FFNN

## Evaluation Framework

- Fold-contained preprocessing
- Stratified 10-fold cross-validation
- Multi-seed evaluation
- Calibration analysis
- Statistical testing (DeLong, McNemar)

## Dataset

Pima Indians Diabetes Dataset  
UCI Machine Learning Repository
